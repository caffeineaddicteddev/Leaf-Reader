## Summary
Improved the book card layout in the library by increasing its height to achieve a more natural portrait aspect ratio for book covers. Fixed a bug where the progress ring remained visible after a book reached the "Ready" state.

## Changes Made
### lib/ui/screens/library/library_screen.dart
- Decreased `childAspectRatio` from `0.68` to `0.62` in the `SliverGrid` delegate. This makes each grid cell taller while keeping the width constant.

### lib/ui/widgets/book_card.dart
- Updated `_BookCover` to accept a `showProgress` boolean parameter.
- Modified `BookCard` to pass `true` to `showProgress` only when the book status is `processing` or `pending`.
- Wrapped the progress ring `Positioned` widget in a conditional check using `showProgress`.
