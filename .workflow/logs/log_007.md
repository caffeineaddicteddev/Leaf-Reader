## Summary
Improved the dark mode aesthetic by transitioning from a flat, high-contrast black/grey scheme to a sophisticated Material 3 "Slate/Zinc" palette. Introduced semantic surface container tokens to provide visual depth and hierarchical layering across all screens.

## Changes Made
### lib/ui/theme/app_theme.dart
- Redefined `darkTheme` with a Zinc-950 (`#09090B`) background.
- Added explicit `surfaceContainer` tokens (`Low`, `Normal`, `High`, `Highest`) ranging from Zinc-900 to Zinc-700.
- Updated `lightTheme` for parity with the new semantic token structure.

### lib/ui/screens/settings/settings_screen.dart
- Refactored section and input field backgrounds to use semantic surface tokens instead of hardcoded conditional colors.
- Removed unused `isDark` variable.

### lib/ui/screens/library/library_screen.dart
- Updated search bar background to `surfaceContainerHigh` for better prominence against the Slate background.

### lib/ui/widgets/book_card.dart
- Updated the book cover placeholder gradient to a theme-aware Slate/Zinc linear gradient.
- Improved contrast for book metadata text.

## Verification
- Verified that component layering is visually distinct in dark mode.
- Verified that light mode remains consistent with previous designs but benefits from the new semantic tokens.
- Resolved "const" compilation errors in `book_card.dart`.
