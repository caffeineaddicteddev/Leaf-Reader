# Bug: Progress ring remains visible after book is ready

## Resolution
The `_BookCover` widget was updated to accept a `showProgress` boolean. This boolean is now only true when the book's status is either `pending` or `processing`. When the status becomes `ready` (or `error`), the progress ring is removed from the UI.
